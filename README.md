# Pore Visualization - 3D Pore Network Analysis## Project OverviewThis project focuses on the **experimental and modeling assessment of the thermal properties of calcium sulfoaluminate cement-based insulating boards utilizing agricultural waste products**.### Research BackgroundWe have developed and analyzed three different thermal insulating boards:- **Thermal Board T1**: Expanded vermiculite + CSA cement + water- **Thermal Board T2**: Expanded vermiculite + rice husk ash + moso bamboo fiber + CSA cement + water - **Thermal Board T3**: Expanded vermiculite + rice husk ash (high quantity) + moso bamboo fiber (high quantity) + CSA cement + water### MethodologyMercury Intrusion Porosimetry (MIP) testing was conducted to understand the pore size distribution in each board. The resulting data is used to create both 2D analytical plots and advanced 3D visualizations of the pore network structure.### Sample Specifications- **Dimensions**: 40×40×160 mm prisms- **Analysis**: Pore size distribution ranging from mesopores to macropores (10 nm - 300 μm)## Features### 2D Analysis- Log differential intrusion vs. pore size plots- Cumulative intrusion analysis- Comparative analysis between thermal boards- Mesopore/macropore classification (100 nm and 2000 nm cutoffs)### 3D Visualization- 3D pore network reconstruction- Volumetric pore distribution modeling- Interactive 3D thermal board representations- Advanced scientific visualization with DEM-like particle systems- Live rendering capabilities- High-performance C++ extensions for complex calculations## Installation### Prerequisites- Python 3.8 or higher- pip package manager### Setup1. **Clone the repository**: `bash   git clone https://github.com/Hetawk/pore-visualization.git   cd pore-visualization   `2. **Install dependencies**: `bash   pip install -r requirements.txt   `3. **Build C++ extensions** (optional, for enhanced performance): `bash   cd app/core/cpp_extensions   python setup.py build_ext --inplace   cd ../../../   `## Usage### Basic Usage1. **Run the main application**: `bash   python launcher.py   `2. **Or run the GUI directly**: `bash   python app/main.py   `### Data RequirementsThe application expects mercury intrusion porosimetry data in CSV format with the following structure:- Columns for each thermal board (T1, T2, T3)- Pore size diameter (nm)- Log differential intrusion (mL/g)## Project Structure`├── app/│   ├── main.py                 #
